# Frontend Design Extractor

Analyzes frontend code and UI screenshots to extract reusable design system elements.

## What This Skill Does

- Extracts color palettes and design tokens from CSS/styled components
- Identifies typography scales, fonts, and text styles
- Documents spacing and layout patterns
- Catalogs UI components and their variants
- Generates design system documentation

## Use Cases

- Reverse-engineering a design system from existing code
- Documenting UI patterns for a style guide
- Extracting tokens before migrating to a new framework
- Analyzing competitor UI for design inspiration
