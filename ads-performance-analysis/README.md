# ads-performance-analysis

## description
Comprehensive diagnostic framework for identifying and resolving e-commerce advertising performance anomalies. Analyzes conversion funnels (impression → click → add-to-cart → order), cross-validates metrics, and maps issues to actionable patterns including high cost ratio, funnel breakdown, traffic quality issues, and bidding inefficiencies.
